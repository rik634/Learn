# Langchain for LLM Application Development
- LangChain is an open source development framework for building LLM applications.
- We have 2 different packages: Python and Javascript (Typescript)
- They are focused on composition and modularity.
- Key value adds:
  a. Modular components
  b. Use cases: Common ways to combine components
- The following are the components of the LangChain:
  1. Models:
     - LLMs: 20+ integrations
     - Chat Models
     - Text Embedding Models: 10+ integrations
  2. Prompts:
     - Prompt Templates
     - Output parsers: 5+ implementations
       - Retry/ fixing logic
     - Example selectors: 5+ implementation
  3. Indexes:
     - Document loaders: 50+ implementations
     - Text splitters: 10+ implementation
     - Vector stores: 10+ implementation
     - Retrievers: 5+ integrations/implementations
  4. Chains:
     - Prompt + LLM + output parsing
     - can be used as building block for longer chains
     - More application specific chains: 20+ types
  5. Agents:
     - Agent Types: 5+ types
       - Algorithms for getting LLMs to use tools
     - Agent Toolkits: 10+ implementations
       - Agents armed with specific tools for a specific application
